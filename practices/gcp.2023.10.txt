Compute Engine: スケーラブルで高性能な仮想マシン。1 つの e2-micro インスタンスが月々無料。
Cloud Storage: 高い性能と信頼性を持ち、様々なストレージ要件に対応。5 GB-月の Standard Storageが無料。
BigQuery: ペタバイト規模のフルマネージド分析データウェアハウス。月々1 TBのクエリが無料。
Google Kubernetes Engine: Googleが管理するKubernetesクラスタを使用して、コンテナオーケストレーションを実施。1つのAutopilotまたはゾーンクラスタが月々無料。
App Engine: スケーラブルなウェブアプリケーションとモバイルバックエンドを構築するプラットフォーム。28インスタンス時間が日々無料。
Cloud Run: ステートレスコンテナを実行するフルマネージド環境。月々200万リクエストが無料。
Cloud Build: 高速で一貫した信頼性の高いビルドをGoogle Cloudで実行。日々120ビルド分が無料。
Operations (旧称 Stackdriver): Google Cloud上のアプリケーションのモニタリング、ロギング、診断。ロギングとモニタリングのための月々の割り当てがあります。
Firestore: アプリのデータを簡単に保存、同期、照会できるNoSQLドキュメントデータベース。1 GBのストレージが無料。
Pub/Sub: リアルタイムで信頼性の高いメッセージングとデータのストリーミングを実現するグローバルサービス。月々10 GBのメッセージが無料。
Cloud Functions: コードによるクラウドサービスの作成と接続が可能なサーバーレス環境。月々200万回の呼び出しが無料。
Vision AI: ラベル検出、OCR、顔検出など多数の機能を提供。月々1,000ユニットが無料。
Speech-to-Text: Google独自のプロダクトにも採用されている音声文字変換。月々60分が無料。
Natural Language API: Googleの機械学習を使用して非構造化テキストから分析情報を引き出す。月々5,000ユニットが無料。
AutoML Tables: 構造化データに基づきMLモデルをビルドしてデプロイ。トレーニングとバッチ予測向けにそれぞれ6ノード時間が無料。
AutoML Natural Language: カスタムMLモデルをトレーニングして、コンテンツを分類。最初の5,000件のテキストレコードと1,000ページのドキュメントページが無料。
AutoML Translation: 翻訳クエリ用のカスタムMLモデルを作成。月々50万文字の翻訳が無料。
Video Intelligence API: 保存された動画とストリーミング動画内の物体、場所、アクションを認識する、事前トレーニングされたMLモデル。月々1,000ユニットが無料。
AutoML Vision: カスタムMLモデルをトレーニングして、画像を一連の独自カテゴリに分類。40ノード時間のトレーニングとオンライン予測が無料。
AutoML Video: カスタムMLモデルをトレーニングして、動画を分類。40ノード時間のトレーニングが無料。
Workflows: Google Cloud や HTTP API でサービス呼び出しのフルマネージドシーケンスを実行。月々5,000の内部ステップが無料。
Cloud Source Repositories: Google CloudがホストするさまざまなプライベートGitリポジトリ。最大5ユーザーが無料で利用可能。
Secret Manager: APIキー、パスワード、証明書、その他の機密データを安全に保存。月々6つのシークレットバージョンが無料。
